## 1. 产品定位

### 一句话描述
> 一个纯前端的 A 股技术指标交互分析工具，支持多标的切换、参数实时调节、图表即时重绘，让量化研究者在不写代码的情况下快速探索 RSI / MACD / 布林带 / ATR / KDJ 五大指标。

### 核心价值
- **零代码探索**：拖拽参数滑块即可看到指标变化，无需改脚本重跑
- **多标的对比**：一键切换中国银行 / 贵州茅台 / 自定义 CSV
- **实时信号**：参数调整后自动重新计算金叉死叉等信号
- **分析就绪**：复权数据直入，计算结果可导出 CSV 供回测使用

### 目标用户
| 用户 | 场景 |
|---|---|
| 量化研究者 | 快速验证指标参数组合，寻找最优参数 |
| 投资分析师 | 交互式查看技术信号，辅助判断 |
| 学习者 | 直观理解各指标参数对形态的影响 |

### 2.2 股票数据选择 (F-01)

#### 内置标的
| 标的 | 代码 | 数据文件 |
|---|---|---|
| 中国银行 | 601988.SH | `data/processed/boc_daily_adj.csv` |
| 贵州茅台 | 600519.SH | `data/processed/moutai_daily_adj.csv` |

#### CSV 上传
- 接受格式：UTF-8 (with BOM) CSV
- 必需字段：`date, open, high, low, close, volume, pct_chg`
- 自动检测分隔符和编码
- 上传后自动校验字段完整性

### 2.3 参数调节 (F-04)

每个参数提供两种输入方式：
- **滑块**：拖拽快速调参，显示当前值
- **数字输入框**：精确输入，支持键盘上下键微调

参数变化后 **300ms 防抖** 触发重绘，避免拖拽过程中频繁计算。

### 2.4 指标预设 (F-06)

| 预设 | RSI | MACD | BOLL | ATR | KDJ |
|---|---|---|---|---|---|
| 标准 (default) | 14 | 12/26/9 | 20/2 | 14 | 9/3/3 |
| 短期 (short_term) | 6 | 5/13/4 | 10/1.5 | 7 | 5/2/2 |
| 长期 (long_term) | 24 | 19/39/9 | 30/2.5 | 21 | 14/3/3 |

### 2.5 时间轴缩放与平移 (F-13 ~ F-19)

所有图表共享同一时间视图区间 (`viewRange: { start, end }`)，任一图表上的缩放/平移操作同步联动所有图表。

#### 交互方式

| 操作 | 方式 | 说明 |
|---|---|---|
| **滚轮缩放** (F-13) | 鼠标滚轮在任意图表上滚动 | 向上滚=放大（缩短可见区间），向下滚=缩小（拉长可见区间），以鼠标 X 位置为缩放中心 |
| **拖拽平移** (F-14) | 鼠标按住图表左右拖拽 | 向左拖=查看更近期数据，向右拖=查看更早期数据 |
| **导航条** (F-15) | 价格图下方迷你导航条 | 拖拽窗口内部=平移，拖拽左右边缘=缩放，点击外部=跳转到该位置 |
| **快捷按钮** (F-16) | 顶栏 1月/2月/3月/半年/1年/全部 | 一键设置可见区间为最近 N 天 |
| **缩放按钮** (F-17) | 每个图表卡片右上角 +/−/⟲ | 以视图中心为基准缩放，⟲ 重置为全部 |
| **X轴标签** (F-18) | 每个图表底部 | 5 个等分位置显示日期 (YYMMDD) |
| **视图信息** (F-19) | 顶栏右侧 | 实时显示 "起始日期 ~ 结束日期 (N 日)" |

#### 技术约束

- 最小可见区间：20 个数据点（避免计算除零和图表空旷）
- 指标计算基于完整数据集，仅显示部分切片（保证 lookback 正确）
- Y 轴范围根据可见区间动态计算（缩放后纵轴自适应）
- 切换标的时自动重置视图到最近 1 年

每个指标为一个卡片，结构：

│ ▬▬▬▬▬●▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬ │  ← 滑块

#### 交互细节
- 开关关闭时卡片半透明 (opacity: 0.4)，对应图表区域隐藏
- 滑块拖拽时实时显示数值，松手后触发重绘
- 预设值按钮点击即切换参数
- 卡片可拖拽排序，调整图表显示顺序

### 3.4 主内容区 — 图表布局

#### 布局规则
1. **主图 (Price + BOLL)**：始终在最上方，高度最大 (320px)
2. **子图**：按侧边栏顺序排列，高度 180px
3. **ATR + KDJ**：可并排显示（各占 50% 宽度）
4. **信号汇总表**：固定在底部
5. 关闭的指标对应图表区域自动隐藏，其余图表上移

#### 图表样式

**主图 (Price + BOLL)**
- K 线图 或 收盘价折线图 (可切换)
- 布林带三轨：上轨虚线 / 中轨实线 / 下轨虚线
- 上下轨之间半透明填充
- 突破上下轨的点用圆圈标记
- 成交量副图（可选，底部 60px）

**RSI 图**
- RSI 折线 (0-100 区间)
- 70 红色虚线 + 30 绿色虚线 + 50 灰色点线
- 超买区域红色半透明背景，超卖区域绿色半透明背景

**MACD 图**
- 柱状图：红正绿负
- DIF 黄线 + DEA 紫线
- 金叉红色 ▲ 标记，死叉绿色 ▽ 标记
- 零轴灰色水平线

**ATR 图**
- TR 浅色柱状图 + ATR 实线
- ATR 峰值标注
- 右侧 Y 轴显示 ATR 百分比

**KDJ 图**
- K 蓝 / D 黄 / J 青 三线
- 80/20 参考线
- 金叉死叉标记

## 4. 技术架构

### 4.1 技术选型

| 层 | 技术 | 说明 |
|---|---|---|
| 前端框架 | 纯 HTML + CSS + JavaScript | 零依赖，单文件部署 |
| 图表库 | 自绘 SVG / Canvas | 轻量，完全可控 |
| 数据处理 | 纯 JS | CSV 解析 + 指标计算 |
| 部署 | 静态文件 | 可直接用浏览器打开 |

> **设计决策**：不使用 React/Vue 等框架，保持单文件零依赖，便于分发和部署。所有计算在浏览器端完成，无需后端服务。

### 4.2 模块划分

```
┌─────────────────────────────────────────┐
│              UI Layer (DOM)              │
│  Header + Sidebar + ChartArea + Table   │
├─────────────────────────────────────────┤
│            Chart Renderer               │
│  PriceChart / RSIChart / MACDChart      │
│  BOLLChart / ATRChart / KDJChart        │
├─────────────────────────────────────────┤
│           Indicator Engine              │
│  RSI / MACD / BOLL / ATR / KDJ          │
│  (纯函数, 输入价格+参数 → 输出指标值)    │
├─────────────────────────────────────────┤
│            Data Manager                 │
│  CSV Parser / DataLoader / Cache        │
├─────────────────────────────────────────┤
│            State Manager                │
│  当前标的 / 时间范围 / 参数 / 开关状态   │
└─────────────────────────────────────────┘
```

### 4.3 数据流

```
CSV文件 ──→ DataLoader.parse() ──→ DataFrame {date[], ohlc[]}
                                        │
                   ┌────────────────────┘
                   ↓
           StateManager.getState()
           {stock, dateRange, indicators: {RSI: {on, params}, ...}}
                   │
                   ↓
           IndicatorEngine.compute(data, params)
           → {RSI: number[], MACD: {DIF, DEA, hist}, ...}
                   │
                   ↓
           ChartRenderer.render(indicatorData, chartConfig)
           → SVG / Canvas 更新
                   │
                   ↓
           SignalDetector.detect(indicatorData)
           → signalTable 更新
```

### 4.4 性能设计

| 关注点 | 方案 |
|---|---|
| 参数拖拽卡顿 | 300ms 防抖 + requestAnimationFrame |
| 大数据量渲染 | Canvas 替代 SVG (数据点 >2000 时) |
| 重复计算 | 参数未变时使用缓存结果 |
| 首次加载 | 内置数据嵌入 HTML (data URI) |

## 5. 指标参数规格

### 5.1 参数定义

| 指标 | 参数 | 类型 | 范围 | 默认 | 步长 |
|---|---|---|---|---|---|
| RSI | period | int | 2-50 | 14 | 1 |
| MACD | fast_period | int | 2-20 | 12 | 1 |
| MACD | slow_period | int | 10-40 | 26 | 1 |
| MACD | signal_period | int | 2-20 | 9 | 1 |
| BOLL | window | int | 5-60 | 20 | 1 |
| BOLL | num_std | float | 1.0-3.5 | 2.0 | 0.1 |
| ATR | period | int | 5-30 | 14 | 1 |
| KDJ | n | int | 5-30 | 9 | 1 |
| KDJ | m1 | int | 2-10 | 3 | 1 |
| KDJ | m2 | int | 2-10 | 3 | 1 |

### 5.2 参数约束

- `MACD.fast_period < MACD.slow_period`（快线必须短于慢线，否则报错提示）
- `BOLL.window >= 5`（窗口太小标准差无意义）
- 所有参数为正整数（`num_std` 除外，支持小数）

| 指标 | 快捷选项 |
|---|---|
| RSI | [6, 12, 14, 24] |
